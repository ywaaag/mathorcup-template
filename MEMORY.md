### 工具调度规则
- **复杂建模** → 自动启动 `sequential-thinking`，至少 3-5 步推理链
- **多模态文件** → 自动调用对应 `document-skills`，禁止直接 cat
- **优化问题** → 优先 `highs` MCP → 其次 OR-Tools，绝不自写低效循环
- **不确定内容** → 先 `fetch` 联网再回答
